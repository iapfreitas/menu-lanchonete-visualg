# 🍔 Sistema de Lanchonete em VisuAlg [ Novembro de 2025 ].

Este é um dos meus primeiros projetos para praticar Lógica de Programação. Trata-se de um sistema de menu para uma lanchonete desenvolvido utilizando o **VisuAlg**.

## 🚀 Funcionalidades
O algoritmo simula o caixa de uma lanchonete permitindo ao usuário:
- Navegar por um menu interativo com opções de Lanches, Acompanhamentos e Bebidas.
- Escolher a quantidade desejada de cada item.
- Calcular o subtotal por categoria (Lanches, Acompanhamentos e Bebidas).
- Exibir o **Valor Total do Pedido** ao finalizar a execução.

## 🧠 O que eu aprendi e apliquei neste projeto:
- **Variáveis e Tipos de Dados:** Uso de variáveis inteiras e reais para armazenar quantidades e preços.
- **Estruturas de Repetição:** Uso do laço `repita ... ate` para manter o menu ativo até que o usuário decida finalizar (digitando 0).
- **Estruturas de Decisão:** Uso de `escolha ... caso` para navegar entre os menus e registrar os pedidos selecionados.
- **Operações Matemáticas:** Lógica para somar os valores cumulativos dos pedidos.

## 🛠️ Como executar
1. Baixe o software [VisuAlg](https://visualg3.com.br/).
2. Faça o download do arquivo `LANCHONETE IARA.ALG` deste repositório.
3. Abra o arquivo no VisuAlg e pressione `F9` para executar o programa.
